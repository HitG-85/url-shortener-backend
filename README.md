URL SHORTENER BACKEND

 ->Brief Task Description

This project is a backend URL shortener service built using Node.js, Express, and MongoDB. It allows users to convert long URLs into short, shareable links and provides analytics on link usage.

As a feature, basic user signup and login functionality has been implemented to demonstrate credential handling and backend API design.




Features:

-Generate short URLs for long links

-Redirect short URLs to the original destination

-Track click analytics (timestamps + total clicks)

-User signup and login functionality


TECH STACK:

-Node.js

-Express.js

-MongoDB

-Mongoose

-nanoid (short URL generation)
